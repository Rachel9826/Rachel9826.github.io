网站导航说明

这是一个空白的网站收藏集，目前所有30个位置都处于待添加状态。

您需要按照下面的"如何编辑网站"说明，手动添加您想要分享的网站。

所有灰色背景的卡片表示该位置待添加内容，添加后会变成彩色并可点击访问。

如何编辑网站？

  1. 右键点击页面，选择"查看页面源代码"或按Ctrl+U
  2. 找到 JavaScript 部分中名为 "websiteData" 的数组（大约在第270行）
  3. 按照以下格式修改网站信息：
  4. { id: 1, name: "网站名称", url: "https://网站地址", category: "分类名称" }
  5. 例如：{ id: 1, name: "Google搜索", url: "https://www.google.com", category: "搜索引擎" }
  6. 修改完成后保存HTML文件，刷新页面即可看到更改
  7. 重复此过程，直到填满所有30个网站位置
